 Question-6For a linear regression problem that has 1000 data-points in the training set in R2, two different optimizers are used to learn the weight vector. One method uses gradient descent and the other uses SGD with a batch size of 20. Both methods are run for 100 iterations using the same learning rate. The initial weight vector is 0 in both cases. The journey of the weight vector in weight-space is given for both methods. The dots correspond to the weight vectors, one for each iteration. The circles correspond to the contours of the loss function. Green: start, Red: stop
 Which of these two images corresponds to gradient descent and which one corresponds to stochastic gradient descent? Explain your answer.
SolutionThe image on the left corresponds to gradient descent and the one on the right corresponds to stochastic gradient descent. In gradient descent, the complete gradient is used. Notice how this results in a movement that is perpendicular to the contours at each iteration. On the other hand, in the case of stochastic gradient descent with batch size of 20 (much less than 1000, the number of data-points), the gradient is incomplete and hence inaccurate. This results in a gradient direction that is not exactly perpendicular to the contours. This manifests in a somewhat zig-zag motion of the weight vector across the iterations and especially near the optimum. However, this direction is not too bad and eventually it converges to roughly the same point as gradient descent.
